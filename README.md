## ng2-preloader ##

Nice preloader for angular 2 apps

![preloader](/screen.png)


import it

    import { PreLoaderComponent } from 'ng2-preloader/ng2-preloader';

include it

    ...
    directives: [PreLoaderComponent]
    ...

use in html

    <pm-pre-loader *ng2if="showIfLoading" ></pm-pre-loader>
